# Домашнее задание

## Теория

---

### **1. Массивы**
Массивы позволяют хранить несколько элементов одного типа в одной переменной.  
Пример создания массива:  
```csharp
int[] numbers = { 1, 2, 3, 4, 5 };
```

Операции с массивами:  
- **Обращение к элементу:**  
  ```csharp
  Console.WriteLine(numbers[0]); // 1
  ```
- **Перебор массива:**  
  ```csharp
  foreach (int number in numbers)
  {
      Console.WriteLine(number);
  }
  ```

---

### **2. LINQ**
**LINQ (Language Integrated Query)** позволяет выполнять запросы к данным.  

Пример фильтрации массива:  
```csharp
int[] numbers = { 1, 2, 3, 4, 5, 6 };

var evenNumbers = numbers.Where(n => n % 2 == 0);

foreach (var n in evenNumbers)
{
    Console.WriteLine(n); // 2, 4, 6
}
```

LINQ поддерживает операции:  
- Фильтрация (`Where`)  
- Сортировка (`OrderBy`)  
- Агрегация (`Sum`, `Average`)  

---

### **3. Методы**
Методы позволяют переиспользовать код.  

Пример метода:  
```csharp
static int Add(int a, int b)
{
    return a + b;
}
```

Методы могут:  
- Возвращать значения с помощью `return`.  
- Принимать параметры: по значению (по умолчанию) или по ссылке (`ref`, `out`).

Пример передачи по ссылке:  
```csharp
static void Increment(ref int number)
{
    number++;
}
```

---

### **4. Конструкция `switch` и перечисления**
Перечисления используются для создания набора констант:  
```csharp
enum Weather { Sunny, Cloudy, Rainy, Snowy }
```

Конструкция `switch` проверяет значение переменной:  
```csharp
Weather today = Weather.Sunny;

switch (today)
{
    case Weather.Sunny:
        Console.WriteLine("Сегодня солнечно!");
        break;
    case Weather.Rainy:
        Console.WriteLine("Идёт дождь.");
        break;
    default:
        Console.WriteLine("Неизвестная погода.");
        break;
}
```

---

### **5. Рекурсия**
Рекурсия — это вызов функции самой себя.  
Пример нахождения суммы чисел:  
```csharp
static int RecursiveSum(int[] array, int length)
{
    if (length == 0)
        return 0;
    return array[length - 1] + RecursiveSum(array, length - 1);
}
```
